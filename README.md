# Website Bordir Komputer AB

## Preview Website

![Preview Website](preview1.png)

## Deskripsi

Website landing page untuk jasa bordir komputer yang melayani pembuatan bordir untuk topi, kaos, jaket, dan seragam.

Website ini dibuat untuk membantu usaha bordir memiliki website sederhana sebagai media promosi online.

## Fitur Website

- Landing page bisnis
- Galeri hasil bordir
- Informasi layanan
- Google Maps lokasi usaha
- Tombol WhatsApp pemesanan

## Teknologi yang digunakan

- HTML
- Tailwind CSS
- JavaScript
- GitHub Pages
